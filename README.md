# RainCheck

This is a work in progress and is not production ready.

If a user sends a text to RainCheck with their zipcode, they will recieve a reply with the hourly weather report for the next 10 hours.

Example Response:

```
09: 48° 0% cloudy

10: 49° 0% cloudy

11: 50° 0% cloudy

12: 50° 3% cloudy

01: 50° 17% cloudy

02: 51° 31% chancerain

03: 51° 42% chancerain

04: 51° 35% chancerain

05: 51° 59% chancerain

06: 50° 61% rain

07: 49° 77% rain

08: 50° 77% chancerain
```
